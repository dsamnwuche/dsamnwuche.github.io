Analysis for New York Citi Bikes

Goal

It is no doubt that data contains the information about the past, present and future of a business. It is always a creed of mine to ask clients the relevant questions, assay the data being given and make the required analysis appropriately. While doing my analysis, I ensure that data cleaning is adequately done, I deduce the descriptive statistics of the dataset before proceeding to develop tables and charts. This ensures that the insight derived from the analysis will help clients make more informed decisions for their businesses.


Result

Results are presented in a report with recommendations and follow-up steps. Clear decisions can be made based on this decisions, sales are maximized and losses minimized.

Project Duration

The project duration was for 2 days. The tables, charts and slides for presentation were developed during this time. After which an evaluation test was given by CareerFoundry.




Questions

  1. What are the most popular pick-up locations across the city for New York Citi Bike rental?
  2. How does the average trip duration vary across different age groups?
  3. Which age group rent the most bikes?
  4. How does bike rental vary across the two user groups (one-time users and long-term subscribers) on different days of the week?
  5. Does user age impact the average bike trip duration?


Software

Microsoft Excel was used in doing this project.



Procedure

  1. I downloaded the dataset from the provided link. Here
  2. Proceeded to clean the data. Firstly, I removed all duplicate entries in the data by highlighting the entire set and clicking on the "data" icon in the toolbar and selecting "remove duplicates". This reduced the number of entries as duplicates were present.
  3. Next, I deleted unwanted rows (blanks) as the presence of missing data may alter the analysis and may give false results. This was done by first creating a filter, using the filter to make blanks in every column perceptible while hiding known parameters. Afterwards, I deleted the entire dataset of the blanks (incomplete rows). This further reduced the dataset markedly, as it ensured that the set was ready for further analysis.
  4. After data cleaning was completed, I proceeded to conduct descriptive statistics on the cleaned dataset, calculating the mean, mode, median, maximum and minimum values of two important variables (trip duration in minutes and user age) from the final question. I employed the following excel formulas;
    • mean: =AVERAGE()
    • median: =MEDIAN()
    • minimum value: =MIN()
    • maximum value: =MAX()
  5. I went on to remove entire rows of outliers (rows of the data with exaggerated values) from the columns (or variables) relevant to the analysis.
  6. I developed pivot tables for the relevant variables to each of the questions asked. Firstly, I started by highlighting all in the dataset, clicking on the "insert" icon in the toolbar and selecting the "PivotTable" operation.
  7. From the pivot table developed, I inputted start station (pick-up location) on the row and count of the bike ID on the column. With the pivot table prepared, I created a bar chart to represent this task. I repeated this sequence for the following tasks; 2,3 and 4.
  8. To answer the fifth question, I developed a scatter plot of the time_duration_in_min against the user age. There was no significance between both variables.
